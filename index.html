Consumer

Comsumers can benefit with this system when multiple users fly drones in the same airspace, and since the drones are interconneted using a decentralized network that does not depend on cell towers, satellites, or wifi, drones can be prevented from colliding with each other even when no other connectivity services are available.

Enterprise

Enterprise can benefit with this system when using multiple drones to perform day to day tasks such as inspection, disaster recovery. You would only need one operator to control multiple drones, and the decentralized network created between drones does not depend on any external connectivity services such as cell towers, satellites, or wifi. As a result, this system would work even when no servies are available such as during hurricanes, or in the middle of no where.

Using multiple drones can benefit enterprise my saving time, personal, and increasing safety by putting less lives in the harms way.
